# 🌐 Personal Portfolio Website

This is my **personal portfolio website** built using **HTML, CSS, and JavaScript**. It showcases my projects, skills, Experience, certificates  and contact information in a clean and professional layout.

---

## 💻 Features
- Fully responsive **portfolio website**  
- Sections for **About Me**, **Projects**, **Skills**, **Certificates**, and **Experience**.  
- Smooth scrolling and interactive **UI/UX effects**  
- Easy to customize and extend  

---

## 🛠 Technologies Used
- **HTML5**  
- **CSS3** (Flexbox, Grid, Animations)  
- **JavaScript** (DOM manipulation, events, interactive elements)
