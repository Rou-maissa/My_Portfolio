# 🌐 Personal Portfolio Website

This is my **personal portfolio website** built using **HTML, CSS, and JavaScript**.  
It showcases my **projects, skills, experience**, and contact info in a clean, professional layout.  

---

## 💻 Features
- Multi-page layout: **Home, Skills, Experience, Projects, Contact**  
- Fully **responsive design** for desktop and mobile  
- Smooth scrolling and **interactive UI/UX effects**  
- Easy to customize and expand  

---

## 🛠 Technologies Used
- **HTML5** – Multi-page structure  
- **CSS3** – Flexbox, Grid, animations, and responsive styling  
- **JavaScript** – DOM manipulation and interactivity
